# cpp_3x3rubikscube
Working representation of a 3x3 rubiks cube using c++ console graphics. 


Instructions in english

W - Turn top layer

A - Turn left layer

S - Turn front layer

D - Turn right layer

X - Turn bottom layer

F - Turn back layer

E - Turn cube perspective in X axis

Y - Turn cube perspective in Y axis

M - Scramble the cube

N - Solve the cube
